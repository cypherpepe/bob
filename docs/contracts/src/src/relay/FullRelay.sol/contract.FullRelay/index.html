Git Source

Inherits: IFullRelay

Author: Distributed Crafts (https://www.gobob.xyz/) Forked from https://github.com/summa-tx/relays Changes made:

  1. dependency changes
  • changed summa-tx/bitcoin-spv to keep-network/bitcoin-spv-sol
  • remove SafeMath
  1. test changes
  • fixed some tests that were written incorrectly in the summa repo
  • ported Truffle javascript tests to Foundry solidity
  • new tests added
  1. solidity compiler version upgraded to 0.8.17
  2. OnDemandSPV was gutted and only the verification part was kept
